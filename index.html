django-deployer

Generic Deployment tool for Django that supports popular PaaS providers such as Heroku, Dotcloud, Stackato, OpenShift, Gondor

Download this project as a .zip file Download this project as a tar.gz file

The goal of django-deployer is to minimize the effort to deploy a Django app to any of the popular PaaS providers. By asking a series of questions about the Django app, we can generate a generic deploy.yml file that can be consumed by any of the PaaS providers. Each PaaS provider can translate these requirements into a configuration that will work to deploy to their PaaS.
